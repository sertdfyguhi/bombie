# bombie
so basically you implement a function called bombie that adds a fullstop at the end of a string and print it using `bombie` like
```js
function bombie(str) {
  return str + '.'
}

console.log(bombie('bombie'))
```
or
```py
def bombie(str):
  return str + '.'

print(bombie('bombie'))
```
